Buffet semanal balanceado

Presentación

El Buffet Semanal Balanceado es una aplicación web que te permite armar visualmente tu menu semanal, para poder tener una alimentacion balanceada. Te ayuda a mantener una alimentación balanceada mostrando que existen diferentes tipos de alimentos que podemos consumir, basados en carnes, pescados, pollos, menestras, pastas, etc. El objetivo es permitirte estar bien alimentado utilizando alimentos naturales que compramos en cualquier supermercado, pudiendo complementarlos con suplmenteos alimentarios si lo consideras necesario.
